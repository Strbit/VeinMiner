# VeinMiner

VeinMiner（连锁挖矿）是一个 TShock 插件

- 原作者：[Megghy](https://github.com/Megghy)
- 我进行了修复使其在新版 tshock 上可用

---

## 命令

- `/vm`  开启或关闭玩家自身的连锁挖矿功能
- `/vm msg`  开启或关闭连锁挖矿时的提示信息

### 配置

- `Enable` - 控制插件是否启用
- `Broadcast` - 是否在玩家进行连锁挖矿时发送挖矿提示
- `PutInInventory` - 是否将挖到的矿石直接放入玩家背包，否则按默认方式掉落（如果背包已满则不会进行连锁挖矿）
- `Tile` - 可进行连锁挖矿的方块 ID 列表
